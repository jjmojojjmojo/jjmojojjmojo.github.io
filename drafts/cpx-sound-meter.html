Application Of State: A CircuitPlayground Express Sound Meter

Tue 26 March 2019
By jjmojojjmojo

WARNING

You are viewing a draft document. It may contain inaccurate, misleading, or unvetted information.

I've run into a bit of a delay in finishing up the State And Events In CircuitPython series. In the meantime, I wanted to do a quick project using some of the concepts and the CPX.

In this tutorial, we'll build a sound meter - using the microphone on the CircuitPlayground Express, we'll show relative sound intensity using the built-in Neopixels. We'll incorporate state and events along the way, and peek into analog signal processing.
